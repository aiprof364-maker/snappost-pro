## **STEP 1: Create Test Facebook Account**

### **1.1 Go to Facebook signup page**
👉 **[https://www.facebook.com/](https://www.facebook.com/)**

### **1.2 Click "Create New Facebook Account"**
- Look for the blue button at the bottom of the login area
- OR go directly: **[https://www.facebook.com/r.php](https://www.facebook.com/r.php)**

### **1.3 Fill in the form**
- **First Name:** Demo
- **Last Name:** Contractor
- **Email:** Use a Gmail or any email you control (e.g., `testcontractor@gmail.com`)
- **Password:** Create a strong password (write it down!)
- **Birthday:** Any date
- **Gender:** Any option

### **1.4 Verify your email**
- Facebook will send a verification code to your email
- Check your email inbox
- Enter the code on Facebook
- ✅ Account created!


### **2.2 Select "Business or Brand"**

### **2.3 Fill in page details**
- **Page Name:** Demo Plumbing Co
- **Category:** Select "Plumbing Service" or "Contractor"

- Click "Create Page"

### **2.4 Complete page setup**
- Add a profile picture (any image)
- Add a cover photo (any image)
- ✅ Business page created!

## **STEP 3: Add Test Account as Tester in Meta Developer Portal**

### **3.1 Go to Meta Developer Portal**
👉 **[https://developers.facebook.com/](https://developers.facebook.com/)**

### **3.2 Navigate to your app**
- Click on "My Apps" (top left)
- Select "SnapPost Pro"

### **3.3 Go to App Roles**
- Left sidebar → **"App roles"**
- OR direct link: **[https://developers.facebook.com/apps/](https://developers.facebook.com/apps/)** → Select SnapPost Pro → **"Roles"** → **"Test Users"**

### **3.4 Add test user**
- Click "Add Test User" button
- **Name:** Demo Contractor
- **Email:** [your test email from Step 1.4]
- **Role:** Tester
- Click "Create Test User"

### **3.5 Set password for test user**
- Meta will show you a temporary password
- Save it (or create your own)
- ✅ Test user added!

## **STEP 5: Record a Screencast**

### **5.1 Download Screencast Tool**
Choose one (all are free):
- **Windows:** [OBS Studio](https://obsproject.com/) or [ScreenFlow](https://www.screenflow.com/)
- **Mac:** [ScreenFlow](https://www.screenflow.com/) or [OBS Studio](https://obsproject.com/)
- **Online:** [Loom](https://www.loom.com/) (easiest - no download needed)

### **5.2 Record the flow**
Using Loom (easiest):
1. Go to **[https://www.loom.com/](https://www.loom.com/)**
2. Click "Start recording"
3. Select "Entire screen"
4. Record yourself doing:
   - Login to SnapPost Pro
   - Upload a photo
   - Generate caption
   - Post to Facebook
   - Show the post on Facebook page
5. Stop recording
6. Get the video link

### **5.3 Save the video link**
```

```
